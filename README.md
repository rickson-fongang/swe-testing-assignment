# Quick-Calc

Quick-Calc is a simple calculator application developed as part of the Advanced Software Engineering course (Lecture 3 — Software Engineering & Testing).

The application implements the four fundamental arithmetic operations (addition, subtraction, multiplication, division) with proper error handling and a clear functionality. The primary focus of this project is not user interface design, but clean architecture, testability, version control discipline, and structured testing strategy.

---

## Features

- Addition
- Subtraction
- Multiplication
- Division (with division-by-zero handling)
- Clear/reset functionality
- Unit and integration test coverage
- Semantic versioning (v1.0.0)
