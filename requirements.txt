## Testing Framework Research

This project compares two popular Python testing frameworks: **pytest** and **unittest**.

### Pytest
Pytest is a modern, widely adopted testing framework known for its concise syntax, powerful fixtures system, and advanced assertion introspection. It allows writing simple test functions without requiring class-based structures. Pytest also provides rich plugin support and better readability for small-to-medium sized projects.

### Unittest
Unittest is Python’s built-in testing framework. It follows a class-based structure inspired by JUnit. While it is stable and fully integrated into Python’s standard library, it often requires more boilerplate code and has less expressive assertion output compared to pytest.

### Justification for Choosing Pytest
Pytest was selected for this project because of its simplicity, readability, and reduced boilerplate. Since this assignment emphasizes clarity, structure, and test design rather than framework complexity, pytest enables a cleaner and more maintainable implementation of both unit and integration tests.
